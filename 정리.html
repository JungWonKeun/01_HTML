엔터 : br

Ch.2 목록관련태그

순서 없는 목록 : ul
순서 있는 목록 : ol
                     - type 속성 : 표시되는 숫자 형식 지정
                      type="a" : 영어 소문자 (a,b,c,d,...)
                     type="A" : 영어 대문자 (A,B,C,D,...)
                     type="i" : 로마 소문자 (i,ii,...,v...x)
                     type="I" : 로마 대문자 (I,II,...,V...X)
                     start 속성 : 목록 시작 번호
                     reversed : 순서 뒤집기(역순,시작부터 -1씩)
목록의 한 줄 : li

자동완성 태그
  • > : 태그 내부에 작성
  • + : 태그 옆에 작성
  • * : * 다음 숫자 만큼 생성
  • {} : 태그 내부에 작성될 내용
  • $ : 1부터 1씩 증가하는 숫자
